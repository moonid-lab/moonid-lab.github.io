## 개요

군집 AI 에이전트(Swarm AI Agent) 기반의 소프트웨어 개발 예측 플랫폼입니다.
다수의 자율 에이전트가 협업하여 개발 주기 전반(일정, 리스크, 품질)을 실시간으로 분석하고 예측합니다.

## 주요 기능

- **Swarm 에이전트 협업** — 역할 분리된 복수의 AI 에이전트가 병렬로 분석 수행
- **개발 일정 예측** — 히스토리 데이터 및 현재 진행 상황 기반 완료 시점 예측
- **리스크 탐지** — 병목 구간, 기술 부채, 의존성 충돌 사전 감지
- **품질 예측** — 코드 복잡도·커버리지·결함 밀도 기반 품질 지표 산출

## 기술 스택

`Python` `LLM` `Multi-Agent` `LLMOps` `RAG` `Kubernetes`

## 연구 분야 연계

- AI Agent Security
- LLMOps / Agentic Memory
- Cloud SecOps
