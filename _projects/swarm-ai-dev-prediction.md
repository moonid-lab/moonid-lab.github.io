## 개요

Development Prediction Platform based-on Swarm AI Agent는
Swarm AI Agent + Ontology + Agentic GraphRAG를 통합하여,
주어진 사건/이벤트의 향후 전개를 예측하고 다양한 개입 시나리오를 실험하는
웹 기반 예측·시뮬레이션 플랫폼입니다.

연구 기간은 2026.04 ~ 2026.11(8개월)이며,
정적 문서 질의응답 중심 RAG를 넘어 동적 행위자 상호작용 기반의 예측형 AI 서비스를 목표로 합니다.

## 연구 배경

- 기존 RAG 시스템은 문서 검색/요약 정확도는 높지만, 사건 전개 과정과 행위자 상호작용을 반영한 미래 예측에는 한계가 있음
- 사회·정책·산업 이슈는 다수 이해관계자와 시간 흐름, 외생 변수로 비선형 전개되므로 시뮬레이션형 접근이 필요
- GraphRAG, 온톨로지 기반 지식구조화, 다중 에이전트 시뮬레이션 연구 동향을 통합해 실증 가능한 플랫폼으로 구현

## 연구 목표

1. 비정형 이벤트 데이터의 도메인 온톨로지 및 지식그래프 자동 구축
2. Agentic GraphRAG 기반 맥락 추론 엔진 개발
3. 성격/목표/기억/상호작용 규칙을 갖는 Swarm AI Agent 시뮬레이션 엔진 개발
4. 예측 보고서 생성 및 결과 설명 기능 개발
5. 검증 가능한 웹 기반 플랫폼/서비스 구현

## 핵심 개발 범위

### 1) 데이터 수집 및 이벤트 표현 체계

- 뉴스, 정책 문서, 보고서, 사용자 제공 자료, SNS 텍스트 등 비정형 데이터 수집
- 사건·행위자·관계·행동·시간·장소·영향요인을 표현하는 이벤트 중심 데이터 모델 설계
- 사건 인과성, 개입 변수, 리스크 요인을 온톨로지 스키마로 구조화

### 2) Ontology + Agentic GraphRAG 엔진

- LLM 기반 엔터티/관계 추출 및 지식그래프 생성 파이프라인 구축
- 관계 추론, 커뮤니티 요약, 시간 흐름 기반 질의응답 지원
- 비정형 문서 메모리를 구조화 그래프로 전환하여 추론 성능 강화

### 3) Swarm AI Agent 시뮬레이션 엔진

- 그래프 상 행위자/집단을 에이전트 객체로 변환
- 각 에이전트에 목표, 성향, 기억, 관계망, 행동 규칙 부여
- 정보 전파, 반응, 협력/갈등, 의사결정 변화의 시간 단계별 시뮬레이션 수행

### 4) 예측 및 시나리오 분석

- 자연어 질의 기반 다중 시나리오 실행
- "개입 A/B 적용 시 결과 변화" 형태의 what-if 분석 지원
- 시나리오별 확률적 결과, 핵심 변수, 민감도, 주요 에이전트 변화 흐름 제공

### 5) 웹서비스 플랫폼

- 데이터 업로드
- 이벤트/그래프 시각화 및 탐색
- 시뮬레이션 실행
- 예측 보고서 조회
- 대화형 질의/해석

### 6) 검증 및 평가

- 실제/준실제 사건 데이터 기반 사례 연구
- 예측 적합도, 시나리오 다양성, 설명 가능성, 질의 정확도, 응답 시간, 사용자 만족도 평가
